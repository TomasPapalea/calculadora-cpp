# Calculadora en C++

Proyecto desarrollado en C++ con el objetivo de practicar los fundamentos de programación y la organización del código mediante funciones.

## Características

* Suma
* Resta
* Multiplicación
* División con validación de división por cero
* Potencias
* Raíces n-ésimas
* Validación de errores en operaciones matemáticas
* Menú interactivo en consola

## Tecnologías utilizadas

* C++
* Biblioteca estándar de C++
* `<cmath>`
* Code::Blocks

## Conceptos aplicados

* Funciones
* Estructuras de control (`if`, `switch`, `while`)
* Validación de datos
* Modularización del código
* Operaciones matemáticas

## Cómo ejecutar

1. Abrir el proyecto en Code::Blocks o cualquier compilador compatible con C++.
2. Compilar el archivo `Calculadora.cpp`.
3. Ejecutar el programa.
